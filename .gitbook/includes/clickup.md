### Choose the ClickUp workspace to connect

![](<../assets/3c3ac9056ae53d4d0eae77689578242a196dd4553edb2d5f20c276f3ac3d38d2 image.png>)

{% step %}
### Set up the integration

Choose the ClickUp workspace you connected in the previous step. Select a space, a list, and an assignee for the integration. You can also choose which information is forwarded to ClickUp.

![](<../assets/095c8f2ff47016ed1d9e43c4d19d6fa5fe1828441bdb07309376d0e70984e840 integrations clickup 2.png>)

![](<../assets/0b6f12a35f9db64cc733bac58a508846ccda3ae0e6f71ca532b4bb7139fea9c7 integrations clickup 3.png>)
{% endstep %}

{% step %}
### Test the integration

Run a test so everything is working smoothly.

![](<../assets/8daf48723eeaf33e928093bfc7556cda5092788946040e115a3b5d4f9ff9953c integrations clickup 4.png>)
{% endstep %}

{% step %}
### Finish setup

Your integration is now set up and ready. From this page you can enable automatic forwarding (this can be reconfigured at any time).

![](<../assets/a016a774f2f8a516028a2834a0d8387373b2e5e6a78d60f5e9c2323c6aafdcd0 integrations clickup 5.png>)
