{% step %}
## Create a webhook in Zapier

Log into your Zapier account and create a new zap, then choose _Webhook_ and then _Catch Hook_.

![Zapier Dashboard](<../assets/caf9003 Zapier_Dashboard.png>)
{% endstep %}

{% step %}
## Add the webhook URL to Luciq

Paste the generated URL into Luciq's integration page. You can also choose which details get forwarded as well.

![Create Zapier - Authenticate](<../assets/04aa5d3c55f729280b7d1d3350ea23b0c5a182a303c407a8a9c7c12a4dbc5d89 integrations zapier 2.png>)
{% endstep %}

{% step %}
## Select products and test

Select the products that you would like to integrate with, whether Bugs & Crashes, APM, or Surveys, and test your integration so that we're sure everything is working smoothly.

![Create Zapier - Test](<../assets/1400b2b5c7c127f7419b2cff758e6a4a6d433f237d5a231901437b54a212f9e3 integrations zapier 3.png>)
{% endstep %}

{% step %}
## Finish setup

All done! Your integration is now set up and ready to go. From this final page, you can allow automatic forwarding (this can be reconfigured at any time).

![Create Zapier - Finish](<../assets/fffc82f16f11cb3c943a2ed2822166cee2cbfc391536016354f0ca1443f691fe integrations zapier 4.png>)
{% endstep %}
{% endstepper %}
