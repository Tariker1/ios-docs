With this integration, we will forward every single bug or issue reported to you into your Shortcut dashboard. So you can easily receive, track and assign issues from Shortcut to your team without getting interrupted.

{% stepper %}
{% step %}
## Enter your Token

Enter your _Token_. To get the token, access your Shortcut app, then go to settings and generate a token.

![](<../assets/db7777d1b127499f72c2a76ec99c9fc220bff6e6d478f148fa49736722bc1fdc integrations shortcut 1.png>)
{% endstep %}

{% step %}
## Choose your Project

Choose your _Project_. You can also choose which details get forwarded as well.

![](<../assets/2bd1440611f243e8c08ef09ef2f181f335fdf2d1b65d675df312de627b02ce94 integrations shortcut 2.png>)
{% endstep %}

{% step %}
## Test the integration

At this point, we just need to test your integration so that we're sure everything is working smoothly.

![](<../assets/45486f391c3d92c336ce98f17650cdc4578288171367bda49545d437af915ce5 integrations shortcut 3.png>)
{% endstep %}

{% step %}
## Finish setup

All done! Your integration is now set up and ready to go. From this final page, you can allow automatic forwarding (don't worry though, this can be reconfigured at any time!)

![](<../assets/c2eedf8ab825fb2ca273608c1d0176d00c81484a5011e44066c0548e60dae84e integrations shortcut 4.png>)
{% endstep %}
