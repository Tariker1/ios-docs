Integrating Luciq with Basecamp helps your team receive all the details they need about issues reported by testers and users from your mobile app using our SDK.

{% stepper %}
{% step %}
## Enter your Company ID

Enter your _Company ID_. You will be redirected to the login page where you can enter your username and password.

![](<../assets/23baa5b82e0ebf5fb82b91736a1ec22f4e7f0ab41daca0d3d7c8ecce026d018d integrations basecamp v2 1.png>)
{% endstep %}

{% step %}
## Allow access

Allow Luciq to access your Basecamp account.

![](<../assets/584a264 2.png>)
{% endstep %}

{% step %}
## Choose Project and Todo List

Choose your _Project_ and _Todo List_. You can also choose which details get forwarded as well.

![](<../assets/c15e34ee0d418c2aeb6d3a2cf5ab33d276e89f9a855ad5bcfd643335a6039f79 integrations basecamp v2 3.png>)
{% endstep %}

{% step %}
## Test the integration

At this point, we just need to test your integration so that we're sure everything is working smoothly.

![](<../assets/9a22c769b5728fc72ddc2c369602e16658bd013d33f2767bea0f065a8ec0009e integrations basecamp v2 4.png>)
{% endstep %}

{% step %}
## Integration complete

All done! Your integration is now set-up and ready to go. From this final page, you can allow automatic forwarding (this can be reconfigured at any time).

![](<../assets/aa62c6d381a2158f17e738cd86810db9e287869d6b8be52987068d2008e9c9e0 integrations basecamp v2 5.png>)
{% endstep %}
